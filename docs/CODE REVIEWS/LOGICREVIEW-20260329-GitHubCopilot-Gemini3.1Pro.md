## Critical Logic Flaws Detailing

### LR-11: Avellaneda-Stoikov (A-S) Reservation Price Saturation (T1-12)
**Severity:** CRITICAL
**Location:** `avellaneda.cpp` (Reservation price equation)

**Context:**
The core equation $r = S - q \cdot \gamma \cdot \sigma^2 \cdot \tau$ requires $q$ (inventory state) to be uniformly scaled with $S$ (Mid Price). 

**The Bug:**
Current logic extracts $S$ as XCH-denominated pricing (double) while computing $q$ as raw base-unit mojos (integer). In the Chia format, $1 XCH = 10^{12} Mojos$. Consequently, $q \cdot \gamma ...$ evaluates to a magnitude $10^{12}$ times larger than the threshold boundary.
This drives $r$ into extreme negative bounds, mathematically forcing all bids generated by the subsequent threshold equation: $bid = \max(0, r - \delta)$ to $0$. The strategy currently creates $0$-priced bids indefinitely under this configuration. 

**Enforced Fix Requirement:**
Apply `double q_converted = static_cast<double>(q) / 1e12;` before piping the state into the A-S evaluation.
