## Key Concepts

### Chia's Unique Features

- **Proof-of-Space-and-Time**: Energy-efficient consensus mechanism
- **Native Offers**: Built-in atomic swap capability
- **Chialisp**: LISP-based smart contract language
- **Non-Custodial**: All trades are trustless and atomic

### Market Making Approaches

1. **AMM Liquidity Provider**: Provide liquidity to pools like TibetSwap
2. **Offer-Based Trading**: Post buy/sell offers on Dexie/OfferBin
3. **Hybrid Strategy**: Combine both approaches for diversification

### Platform Options

- **TibetSwap**: Leading AMM (like Uniswap for Chia)
- **Dexie**: Most popular offer aggregator
- **OfferBin**: Community-driven P2P exchange
- **Cross-Chain**: Bridge to other chains via wXCH
